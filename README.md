## 📖 커리큘럼

| 단계 | 폴더 | 주제 | 핵심 개념 |
|:----:|------|------|-----------|
| 1 | `html` | HTML | 구조, 태그, 미디어, 폼 |
| 2 | `css` | CSS | 선택자, 박스모델, 레이아웃, 스타일링 |
| 3 | `javascript` | JavaScript | 변수, 함수, DOM 조작, 이벤트 |
| 4 | `react` | React | 컴포넌트, props, state, 훅(Hook) |
