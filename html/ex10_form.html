input 태그의 그 외 type 속성

password

number : 숫자 입력

email : 이메일 입력

date : 날짜 입력

color : 색상표 선택

hidden : 값만 존재. 화면에 보이지 않음

file : 파일선택 창. 파일 업로드시 파일 선택

button : 버튼. 이벤트 처리로 기능설정

submit : 버튼. form 태그에 입력된 값들을 서버로 전송

reset : 버튼. form 태그에 입력된 값을 초기화

radio : 단일 선택. name 속성값 기준으로 단일 선택됨

남 여 합격 불합격

check : 다중 선택

축구 요리 여행

fieldest : 입력요소들의 그룹화
