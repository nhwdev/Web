자바 스크립트 개요

HTML페이지의 동작을 사용자의 요구에 맞도록 실행
브라우저의 언어(클라이언트 언어)
인터프리터 방식의 언어
// HTML의 주석은 아님. 자바스크립에서 주석.

자바스크립의 특징

  • 접근제한자 개념 없음
  • 변수 선언이 필요 없음. 초기화될때 자료형이 결정됨.
  • 웹 환경에서 사용되는 언어. => 보안등의 이유로 제한된 기능을 가짐
  • 보안에 취약함: 소스가 외부로 공개됨.

자바스크립트 방식

  • 인라인(inline): 태그내에 간단히 코드 추가
  • 내부(internal): html 파일의 script 태그 내에 코드 추가
  • 외부(external): 별도의 *.js 파일로 코드 구현. 파일을 연결하여 사용하는 방식

인라인 방식

<태그 on이벤트명='자바스크립트소스'>
<a href='javascript:자바스크립트소스'>문장<a>



a 태그에서 알림창 출력
a 태그에서 콘솔 출력

내부방식(internal)

html 페이지내에 script 태그 영역 내부에서 처리되는 소스
script 태그의 위치는 어디든지 가능함. 단 html 태그 내부에 작성하는 것을 권장함

외부방식(external)

html 파일 외부의 .js 파일에 작성된 자바스크립트 파일을 사용함
.js 파일은 <script src="js파일의위치"></script> 형식으로 설정함
