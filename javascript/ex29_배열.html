배열

배열이란. 동일한 자료형을 여러개 저장할 수 있는 자료형을 의미함
JavaScript에서는 자료형에 상관없이 배열에 저장 할 수 있음
javaScript에서는 크기에 제한이 없다. 가변적임
자바의 ArrayList와 더 가깝다.
[] 표기법을 사용함

배열에 값을 저장하기

배열의 함수

배열.indexOf(요소) : 배열 내의 요소의 위치(인덱스) 값 리턴. 없는 경우 -1 리턴

배열.concat(배열,배열) : 배열에 배열 추가하기

배열.reverse() : 배열의 요소를 역순으로 변경. 배열 자체가 변경

배열.sort() : 배열의 요소를 순서대로 정렬. 배열 자체가 변경

배열.push(), pop() : 배열의 요소를 추가, 조회

push(요소값) : 배열에 끝에 인덱스,요소값을 추가. 배열의 크기를 리턴
pop() : 배열의 마지막 요소를 리턴. 배열에서 제거

배열.unshift(), shift() : 배열의 요소를 맨 앞에 추가, 조회

unshift(요소값) : 배열에 맨 앞에 요소값을 추가. 배열의 크기를 리턴
shift() : 배열의 맨 앞 요소를 리턴. 배열에서 제거

배열.slice() : 배열의 부분 배열

배열.splice() : 배열의 요소를 제거 및 추가

- 배열.slice(시작인덱스, 종료인덱스+1) : 시작인덱스 부터 종료인덱스 까지 리턴
- 배열.splice(시작인덱스, 제거갯수, [추가할값]) : 시작인덱스 부터 갯수까지 제거 후 추가
