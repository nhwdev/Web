BOM(Browser Object Mode)

        window 객체 : 자바스크립트의 최상위 객체. 생략가능
            ↪ location 객체 : 주소창과 관련 있는 객체
            ↪ screen 객체 : 사용 모니터 정보
            ↪ navigator 객체 : 브라우저 정보
            ↪ history 객체 : 페이지 방문 기록
            ↪ document 객체 : 문서 정보

location 객체

navigator 객체

history 객체

브라우저에서 방문했던 페이지 기록 정보


        history.bakc() : 이전 페이지로 이동
        history.forward() : 다음 페이지로 이동
        history.go(1) : 다음페이지로 이동. forward와 동일
        history.go(-1) : 이전페이지로 이동. back와 동일

DOM(Document Object Model)

        document
            ↪ html
                ↪ head
                    ↪ meta
                    ↪ title
                ↪ body
                    ↪ div
                    ↪ ...
  • html의 각 요소들을 노드(Node)라고 함
  • 노드의 종류
    • 요소노드(Element Node) : 태그 자체
    • 텍스트 노드(Text Node) : 태그내에 기록된 내용
    • 텍스트 노드가 존재하는 요소 : div, ul, li, ... <시작태그>텍스트노드
    • 텍스트 노드가 없는 요소(Empty 태그) : input, img ... <시작태그 ... />
